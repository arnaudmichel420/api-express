# Test sequalize

## Installation

```
npm install
```

## Remplir les variables d'environemment

Changer le config.json dans /config

## Lancé les migrations

```
npx sequelize-cli db:migrate
```

## Lancé les seeders

```
npx sequelize-cli db:seed:all
```

## Démarrer le serveur

```
npm start
```

## Build le projet

```
npm build
```
